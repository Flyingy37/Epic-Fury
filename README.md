# תרגול DATEDIF - Streamlit App


## מה יש באפליקציה
- תצוגה של המצגת או התוכן מתוך `paste.txt`
- הורדה של קובץ התרגול
- הטמעה של Google Sheet
- לוגו של המגמה ועיצוב RTL לעברית

## קבצים חשובים
- `app.py`
- `logo.gif`
- `paste.txt`
- `תרגול_DATEDIF.xlsx`
- `requirements.txt`

## הרצה מקומית
```bash
streamlit run app.py
```

## העלאה ל Streamlit Community Cloud
1. מעלים את כל תוכן התיקייה ל GitHub
2. בוחרים את ה repository בתוך Streamlit Community Cloud
3. מגדירים את `app.py` כקובץ הראשי
4. לוחצים Deploy

## הערה
אם Google Sheet לא מוצג בתוך האפליקציה, יש לוודא שהמסמך משותף לצפייה באמצעות קישור.
